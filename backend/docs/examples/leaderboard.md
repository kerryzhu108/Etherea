# Get Leaderboard data
Return a list of all player ids, firstnames, lastnames, and experience points

**URL** : `/leaderboard/`

**Method** : `GET`

**Auth** : `NO`

**Data Example**

```json
{
    "leaderboard": [{"uid":11, "name":"Zachary Vincze", "exp":0}, {"uid":13, "name":"Jack Jackson", "exp":15}]
}
```

## Success Response
**Code** : `200 OK`
